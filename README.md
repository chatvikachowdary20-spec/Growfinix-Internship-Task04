# Text-to-Speech & Audio Generation

## Overview

This project converts long-form text such as blog posts or news articles into natural-sounding speech using the OpenAI Text-to-Speech API.

## Features

- Converts text into studio-quality audio
- Automatic long-text chunking
- Generates multiple MP3 files
- Clean and modular Python code
- Easy to customize

## Technologies

- Python
- OpenAI API

## Installation

```bash
pip install -r requirements.txt
```

Add your API key inside `app.py`.

Run

```bash
python app.py
```

Generated audio files will be saved inside the `output` folder.
